HTML Meta Tags

Meta tags provide metadata about your HTML document.

Essential Meta Tags

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta name="description" content="Free web development tutorials"> <meta name="keywords" content="HTML, CSS, JavaScript"> <meta name="author" content="Your Name"> <title>Page Title</title> </head>

Open Graph Meta Tags (for Social Media)

<meta property="og:title" content="Your Page Title"> <meta property="og:description" content="Your page description"> <meta property="og:image" content="thumbnail.jpg"> <meta property="og:url" content="https://yourwebsite.com/page"> <meta property="og:type" content="website">

Additional Useful Meta Tags

<!-- Prevent translation --> <meta name="google" content="notranslate"> <!-- Theme color for mobile browsers --> <meta name="theme-color" content="#ed8936"> <!-- Refresh page every 30 seconds --> <meta http-equiv="refresh" content="30"> <!-- Prevent indexing --> <meta name="robots" content="noindex, nofollow">
SEO Tip: Proper meta tags help search engines understand your content and improve your site's visibility.

Advanced HTML5 Features

HTML5 introduced many new elements and features for modern web development.

Details & Summary Elements

<details> <summary>Click to expand</summary> <p>This is hidden content that appears when you click the summary.</p> <p>You can put any HTML content here!</p> </details>
What is HTML5?

HTML5 is the latest version of HTML with new elements, attributes, and behaviors for richer web applications.

Why use Semantic HTML?

Semantic HTML improves accessibility, SEO, and makes your code easier to maintain.

HTML Accessibility

Making your website accessible to all users, including those with disabilities.

ARIA Roles and Attributes

<!-- Landmark roles --> <header role="banner"> <nav role="navigation"> <main role="main"> <aside role="complementary"> <footer role="contentinfo"> <!-- Button with ARIA --> <button aria-label="Close dialog" aria-expanded="false" aria-controls="dialog1"> X </button> <!-- Live region for dynamic content --> <div aria-live="polite" id="notifications"></div>

Accessible Forms

<form> <fieldset> <legend>Personal Information</legend> <label for="name">Full Name:</label> <input type="text" id="name" name="name" aria-required="true" aria-describedby="name-help"> <div id="name-help" class="help-text"> Please enter your full legal name </div> <label for="email">Email:</label> <input type="email" id="email" name="email" aria-required="true"> </fieldset> </form>

Skip Navigation Link

<!-- Skip to main content link --> <a href="#main-content" class="skip-link"> Skip to main content </a> <nav> <!-- Navigation menu --> </nav> <main id="main-content"> <!-- Main content --> </main> <style> .skip-link { position: absolute; top: -40px; left: 6px; background: #000; color: white; padding: 8px; text-decoration: none; } .skip-link:focus { top: 6px; } </style>
Accessibility First: Always test your website with screen readers and keyboard navigation to ensure it's accessible to all users.
