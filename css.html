CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages.

How CSS Works

CSS consists of style rules that the browser interprets and applies to elements:

/* Selector { Property: Value; } */ h1 { color: blue; font-size: 24px; margin-bottom: 20px; }

Ways to Add CSS

1. Inline Styles

<h1 style="color: red; font-size: 24px;">Hello World</h1>

2. Internal Stylesheet

<style> h1 { color: blue; font-size: 24px; } </style>

3. External Stylesheet

<link rel="stylesheet" href="styles.css">
Best Practice: Use external stylesheets for better organization and maintainability.

CSS Selectors

Selectors target HTML elements to apply styles.

Basic Selectors

/* Element Selector */ p { color: #333; } /* Class Selector */ .highlight { background-color: yellow; } /* ID Selector */ #header { background: #f8f9fa; } /* Universal Selector */ * { margin: 0; padding: 0; }

Advanced Selectors

/* Descendant Selector */ article p { line-height: 1.6; } /* Child Selector */ ul > li { list-style: square; } /* Adjacent Sibling */ h2 + p { margin-top: 0; } /* Attribute Selector */ input[type="text"] { border: 1px solid #ccc; } /* Pseudo-classes */ a:hover { color: red; } button:active { transform: scale(0.98); } /* Pseudo-elements */ p::first-line { font-weight: bold; }

CSS Box Model

Every element is a rectangular box with content, padding, border, and margin.

.box { /* Content */ width: 300px; height: 200px; /* Padding - space inside */ padding: 20px; /* Border - edge around */ border: 2px solid #333; /* Margin - space outside */ margin: 30px; /* Total width: 300 + 40 + 4 + 60 = 404px */ }

Box-Sizing Property

/* Traditional box model */ .box-old { width: 300px; padding: 20px; border: 2px solid #000; /* Total width: 300 + 40 + 4 = 344px */ } /* Modern box model */ .box-new { width: 300px; padding: 20px; border: 2px solid #000; box-sizing: border-box; /* Total width: 300px (includes padding and border) */ }
Tip: Use box-sizing: border-box for more predictable layouts.
