CSS Flexbox

Flexbox is a one-dimensional layout method for arranging items in rows or columns.

Flex Container Properties

.container { display: flex; flex-direction: row; /* row | row-reverse | column | column-reverse */ flex-wrap: nowrap; /* nowrap | wrap | wrap-reverse */ justify-content: flex-start; /* flex-start | flex-end | center | space-between | space-around | space-evenly */ align-items: stretch; /* stretch | flex-start | flex-end | center | baseline */ align-content: stretch; /* For multi-line flex containers */ gap: 10px; /* Space between items */ }

Flex Item Properties

.item { order: 0; /* Change visual order */ flex-grow: 0; /* Ability to grow if necessary */ flex-shrink: 1; /* Ability to shrink if necessary */ flex-basis: auto; /* Default size before growing/shrinking */ align-self: auto; /* Override container's align-items */ }

Common Flexbox Patterns

/* Center vertically and horizontally */ .center { display: flex; justify-content: center; align-items: center; } /* Space items evenly */ .even-spacing { display: flex; justify-content: space-between; } /* Responsive navigation */ .nav { display: flex; flex-wrap: wrap; gap: 1rem; } /* Card layout */ .cards { display: flex; flex-wrap: wrap; gap: 20px; } .card { flex: 1 1 300px; /* Grow, shrink, basis */ }

CSS Grid

CSS Grid is a two-dimensional layout system for the web.

Grid Container Properties

.container { display: grid; grid-template-columns: 1fr 1fr 1fr; /* Track sizes */ grid-template-rows: 100px auto 100px; /* Row sizes */ gap: 20px; /* Space between items */ grid-template-areas: "header header header" "sidebar main main" "footer footer footer"; }

Grid Item Properties

.item { grid-column: 1 / 3; /* Start / end */ grid-row: 1; /* Row position */ grid-area: header; /* Named area */ justify-self: stretch; /* Horizontal alignment */ align-self: stretch; /* Vertical alignment */ }

Advanced Grid Features

/* Responsive grid with auto-fit */ .responsive-grid { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 20px; } /* Named grid lines */ .named-grid { display: grid; grid-template-columns: [start] 1fr [content-start] 2fr [content-end] 1fr [end]; } /* Grid with masonry layout */ .masonry { display: grid; grid-template-columns: repeat(3, 1fr); grid-auto-rows: 100px; gap: 10px; }
Tip: Use grid-template-areas for visual layout planning and easier maintenance.

Responsive Design

Create layouts that work on all screen sizes using media queries and flexible units.

Media Queries

/* Mobile First Approach */ .container { padding: 10px; } /* Tablet */ @media (min-width: 768px) { .container { padding: 20px; max-width: 720px; margin: 0 auto; } } /* Desktop */ @media (min-width: 1024px) { .container { max-width: 1200px; display: grid; grid-template-columns: 1fr 3fr; } } /* Large screens */ @media (min-width: 1440px) { .container { max-width: 1400px; } }

Responsive Units

/* Relative units */ .relative { font-size: 1rem; /* Relative to root */ padding: 1em; /* Relative to element's font size */ width: 50%; /* Relative to parent */ height: 50vh; /* Relative to viewport height */ max-width: 100vw; /* Relative to viewport width */ } /* Container queries (modern) */ @container (min-width: 400px) { .card { display: grid; grid-template-columns: 1fr 2fr; } }

Flexible Layout Patterns

/* Responsive grid */ .grid-responsive { display: grid; grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)); gap: 1rem; } /* Flexible navigation */ .nav-responsive { display: flex; flex-wrap: wrap; gap: 1rem; } @media (max-width: 768px) { .nav-responsive { flex-direction: column; } } /* Responsive typography */ h1 { font-size: clamp(2rem, 5vw, 4rem); }

CSS Positioning

Control how elements are positioned in the document flow.

Position Properties

.static { position: static; /* Default - in normal flow */ } .relative { position: relative; /* In normal flow, can offset */ top: 10px; left: 20px; } .absolute { position: absolute; /* Relative to nearest positioned ancestor */ top: 0; right: 0; } .fixed { position: fixed; /* Relative to viewport */ bottom: 20px; right: 20px; } .sticky { position: sticky; /* Relative until scrolled past */ top: 0; }

Z-index and Stacking Context

.modal { position: fixed; top: 50%; left: 50%; transform: translate(-50%, -50%); z-index: 1000; /* Higher z-index appears on top */ } .backdrop { position: fixed; top: 0; left: 0; width: 100%; height: 100%; background: rgba(0,0,0,0.5); z-index: 999; }

Practical Examples

/* Sticky header */ .header { position: sticky; top: 0; background: white; z-index: 100; } /* Tooltip */ .tooltip { position: relative; } .tooltip::after { content: attr(data-tooltip); position: absolute; bottom: 100%; left: 50%; transform: translateX(-50%); background: #333; color: white; padding: 5px 10px; border-radius: 4px; white-space: nowrap; }
Warning: Overusing absolute and fixed positioning can make layouts brittle. Prefer Flexbox and Grid for most layouts.
