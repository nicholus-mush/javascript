JavaScript Basics

JavaScript is a programming language that adds interactivity to your website.

What Can JavaScript Do?

  • Manipulate HTML content
  • Change CSS styles dynamically
  • Validate form data
  • Create animations and effects
  • Handle user interactions
  • Make API calls

Variables & Constants

Variables are containers for storing data values.

Variable Declaration

// var (function-scoped, older way) var oldVariable = "I'm old school"; // let (block-scoped, modern) let changeable = "I can change"; changeable = "Now I'm different"; // const (block-scoped, cannot be reassigned) const constantValue = "I cannot change"; // constantValue = "This would cause an error"; // Multiple variables let firstName = "John", lastName = "Doe", age = 30;

Variable Naming Rules

  • Names can contain letters, digits, underscores, and dollar signs
  • Names must begin with a letter, $, or _
  • Names are case sensitive (age and Age are different)
  • Reserved words cannot be used as names

Good vs Bad Variable Names

// Good variable names let userName = "John"; let itemCount = 5; let isLoggedIn = true; // Bad variable names let u = "John"; // Too vague let itmcnt = 5; // Hard to read let log = true; // Unclear meaning

Constants and Scope

// Global scope const PI = 3.14159; function calculateArea(radius) { // Function scope let area = PI * radius * radius; return area; } { // Block scope let temporary = "I only exist in this block"; console.log(temporary); } // console.log(temporary); // Error - temporary doesn't exist here

const

Functions

Functions are reusable blocks of code that perform specific tasks.

Function Declaration

// Function declaration function greet(name) { return "Hello, " + name + "!"; } // Function expression const calculateArea = function(radius) { return Math.PI * radius * radius; }; // Arrow function (ES6+) const multiply = (a, b) => a * b; // Calling functions console.log(greet("Alice")); // "Hello, Alice!" console.log(calculateArea(5)); // 78.5398... console.log(multiply(4, 5)); // 20

Parameters and Arguments

// Function with parameters function introduce(name, age, city) { return `My name is ${name}, I'm ${age} years old, and I live in ${city}.`; } // Calling with arguments console.log(introduce("Bob", 30, "New York")); // Default parameters function createUser(name, role = "user") { return { name: name, role: role }; } console.log(createUser("Alice")); // {name: "Alice", role: "user"} console.log(createUser("Bob", "admin")); // {name: "Bob", role: "admin"}

Return Values

// Function that returns a value function isAdult(age) { if (age >= 18) { return true; } else { return false; } } // Function without return (returns undefined) function sayHello() { console.log("Hello!"); } // Multiple return points function getGrade(score) { if (score >= 90) return "A"; if (score >= 80) return "B"; if (score >= 70) return "C"; return "F"; }

Control Flow

Control the flow of your program with conditional statements and loops.

Conditional Statements

// if statement let temperature = 25; if (temperature > 30) { console.log("It's hot outside!"); } else if (temperature > 20) { console.log("The weather is pleasant."); } else { console.log("It's cold outside!"); } // switch statement let day = "Monday"; switch (day) { case "Monday": console.log("Start of the work week"); break; case "Friday": console.log("Weekend is coming!"); break; default: console.log("It's a regular day"); } // Ternary operator let age = 20; let canVote = age >= 18 ? "Yes" : "No"; console.log(canVote); // "Yes"

Loops

// for loop for (let i = 0; i < 5; i++) { console.log("Iteration: " + i); } // while loop let count = 0; while (count < 3) { console.log("Count: " + count); count++; } // do-while loop let number = 5; do { console.log("Number: " + number); number--; } while (number > 0); // for...of loop (arrays) const fruits = ["apple", "banana", "orange"]; for (const fruit of fruits) { console.log(fruit); }

Break and Continue

// break - exit loop early for (let i = 0; i < 10; i++) { if (i === 5) { break; // Exit loop when i is 5 } console.log(i); // 0, 1, 2, 3, 4 } // continue - skip current iteration for (let i = 0; i < 5; i++) { if (i === 2) { continue; // Skip when i is 2 } console.log(i); // 0, 1, 3, 4 }
