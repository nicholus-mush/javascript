CSS Transitions

Transitions allow you to change property values smoothly over a given duration.

Transition Properties

.element { transition-property: all; /* Properties to animate */ transition-duration: 0.3s; /* How long transition takes */ transition-timing-function: ease; /* Speed curve */ transition-delay: 0s; /* When transition starts */ /* Shorthand */ transition: all 0.3s ease 0s; }

Timing Functions

/* Predefined timing functions */ transition-timing-function: ease; transition-timing-function: ease-in; transition-timing-function: ease-out; transition-timing-function: ease-in-out; transition-timing-function: linear; transition-timing-function: steps(4); /* Custom cubic-bezier */ transition-timing-function: cubic-bezier(0.68, -0.55, 0.265, 1.55);

Practical Examples

/* Button hover effect */ .btn { background: #667eea; transition: all 0.3s ease; } .btn:hover { background: #764ba2; transform: translateY(-2px); box-shadow: 0 4px 8px rgba(0,0,0,0.2); } /* Card reveal */ .card { opacity: 0; transform: translateY(20px); transition: all 0.5s ease; } .card.visible { opacity: 1; transform: translateY(0); }

CSS Animations

Animations allow you to create complex, multi-step transitions using keyframes.

Keyframes Syntax

@keyframes slideIn { from { transform: translateX(-100%); opacity: 0; } to { transform: translateX(0); opacity: 1; } } /* Or use percentages */ @keyframes bounce { 0% { transform: translateY(0); } 50% { transform: translateY(-30px); } 100% { transform: translateY(0); } }

Animation Properties

.element { animation-name: slideIn; animation-duration: 1s; animation-timing-function: ease; animation-delay: 0s; animation-iteration-count: 1; /* infinite */ animation-direction: normal; /* reverse | alternate | alternate-reverse */ animation-fill-mode: none; /* forwards | backwards | both */ animation-play-state: running; /* paused */ /* Shorthand */ animation: slideIn 1s ease 0s 1 normal none running; }

Advanced Animation Techniques

/* Multiple animations */ .element { animation: slideIn 1s ease, fadeIn 0.5s ease 0.5s both; } /* Animation with steps */ .typing { animation: typing 3s steps(40) infinite; } /* Pause on hover */ .element:hover { animation-play-state: paused; }

CSS Transforms

Transforms allow you to rotate, scale, skew, or translate elements.

2D Transforms

.element { /* Translate - move element */ transform: translateX(50px); transform: translateY(-20px); transform: translate(50px, -20px); /* Scale - resize element */ transform: scale(1.5); transform: scaleX(2); transform: scale(1.2, 0.8); /* Rotate - rotate element */ transform: rotate(45deg); /* Skew - distort element */ transform: skew(15deg); transform: skew(15deg, 10deg); /* Multiple transforms */ transform: translate(50px, 50px) rotate(45deg) scale(1.2); }

3D Transforms

.container { perspective: 1000px; /* 3D perspective */ } .element { /* 3D Transforms */ transform: rotateX(45deg); transform: rotateY(45deg); transform: rotateZ(45deg); transform: rotate3d(1, 1, 1, 45deg); transform: translateZ(100px); transform: translate3d(50px, 50px, 50px); /* Transform origin */ transform-origin: center center; transform-origin: 0 0; transform-origin: 100% 100%; }

New CSS features that make styling more powerful and efficient.

CSS Custom Properties (Variables)

:root { --primary-color: #667eea; --secondary-color: #764ba2; --spacing: 20px; --border-radius: 8px; } .element { background: var(--primary-color); padding: var(--spacing); border-radius: var(--border-radius); } /* Fallback values */ .element { color: var(--custom-color, #333); }
