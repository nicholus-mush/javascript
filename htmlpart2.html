HTML Tables

Tables are used to display tabular data in rows and columns.

Basic Table Structure

<table> <tr> <th>Header 1</th> <th>Header 2</th> <th>Header 3</th> </tr> <tr> <td>Row 1, Cell 1</td> <td>Row 1, Cell 2</td> <td>Row 1, Cell 3</td> </tr> <tr> <td>Row 2, Cell 1</td> <td>Row 2, Cell 2</td> <td>Row 2, Cell 3</td> </tr> </table>

Result:

Header 1 Header 2 Header 3
Row 1, Cell 1 Row 1, Cell 2 Row 1, Cell 3
Row 2, Cell 1 Row 2, Cell 2 Row 2, Cell 3

Advanced Table Features

<table> <caption>Employee Information</caption> <thead> <tr> <th scope="col">Name</th> <th scope="col">Department</th> <th scope="col">Salary</th> </tr> </thead> <tbody> <tr> <td>John Doe</td> <td>Engineering</td> <td>$75,000</td> </tr> <tr> <td>Jane Smith</td> <td>Marketing</td> <td>$65,000</td> </tr> </tbody> <tfoot> <tr> <td colspan="2">Total</td> <td>$140,000</td> </tr> </tfoot> </table>
Best Practice: Use thead, tbody, and tfoot for better structure and accessibility.

HTML Forms

Forms allow users to input data and interact with your website.

Basic Form Structure

<form action="/submit" method="POST"> <label for="name">Name:</label> <input type="text" id="name" name="name" required> <label for="email">Email:</label> <input type="email" id="email" name="email" required> <label for="password">Password:</label> <input type="password" id="password" name="password" required> <button type="submit">Submit</button> </form>

Form Input Types

<!-- Text Input --> <input type="text" placeholder="Enter text"> <!-- Email Input --> <input type="email" placeholder="Enter email"> <!-- Password Input --> <input type="password" placeholder="Enter password"> <!-- Number Input --> <input type="number" min="1" max="100"> <!-- Date Input --> <input type="date"> <!-- Checkbox --> <input type="checkbox" id="subscribe"> <label for="subscribe">Subscribe to newsletter</label> <!-- Radio Buttons --> <input type="radio" id="male" name="gender"> <label for="male">Male</label> <input type="radio" id="female" name="gender"> <label for="female">Female</label> <!-- Select Dropdown --> <select> <option value="">Select option</option> <option value="1">Option 1</option> <option value="2">Option 2</option> </select> <!-- Textarea --> <textarea rows="4" placeholder="Enter message"></textarea>

Semantic HTML

Semantic HTML uses meaningful tags that clearly describe their purpose.

Why Use Semantic HTML?

  • Better accessibility for screen readers
  • Improved SEO
  • Easier to maintain and understand
  • Better structure for CSS styling

Common Semantic Tags

<header> <h1>Website Header</h1> <nav> <a href="/">Home</a> <a href="/about">About</a> <a href="/contact">Contact</a> </nav> </header> <main> <article> <h2>Article Title</h2> <p>Article content...</p> <section> <h3>Section Title</h3> <p>Section content...</p> </section> </article> <aside> <h3>Related Links</h3> <ul> <li><a href="#">Link 1</a></li> <li><a href="#">Link 2</a></li> </ul> </aside> </main> <footer> <p>© 2023 My Website</p> </footer>
